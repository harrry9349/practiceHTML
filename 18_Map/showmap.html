Googleの地図を埋め込む

  1. 埋め込みたい場所を選択する
  2. [共有] をクリックする。
  3. [地図を埋め込む] をクリックする。
  4. 表示されている HTML コードをコピーする。
  5. ウェブサイトの HTML にコードを貼り付ける。

Yahooの地図を埋め込む

  1. 埋め込みたい場所を選択する
  2. [URL] をクリックする。
  3. 「この地図をブログ、サイトにはりつける」の下に表示されている HTML コードをコピーする。
  4. ウェブサイトの HTML にコードを貼り付ける。
